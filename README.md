# Sports-Club-Management-Web-Application
A web platform to centralize and streamline club operations, including member management, activities, pricing, and internal roles. Improves administrative efficiency and provides a better experience for members.

Features

Member management with profiles, categories, and attendance tracking

Role-based access: admins and staff with appropriate permissions

Activity scheduling for trainings, tournaments, and events

Pricing management for memberships and activities

Secure login and password hashing

Automated tracking and reporting

Deployed on XAMPP for stable operations

Impact

Reduced administrative workload by ~30%

Improved data accuracy and reduced errors by ~25%

Enabled real-time updates and seamless internal communication

Tech Stack

Backend: PHP

Frontend: JavaScript

Database: MySQL

Deployment: XAMPP
